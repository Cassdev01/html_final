Alinhando a Alimentação com os Treinos na Academia
Dev: Cassiano Gomes

O objetivo principal deste site é ensaiar rotinas de programação desenvolvendo um site do zero, utilizando a linguagem de marcação HTML em seu desenvolvimento. O conteudo irá ensinar como desenvolver o hábito de uma alimentação saudável, alinhando esta rotina com os treinos de academia.


Treinar e fazer dieta parece fácil, mas fazer ambos corretamente e obter resultados é algo que vai bem mais além.

Aqui iremos ajudar voce a alcançar e superar resultados!

Os conteudos serão dispostos em tópicos, formando um índice agrupados com links clicaveis, para que em caso de dúvidas, possa se obter um detalhamento maior no asssunto de interesse.
