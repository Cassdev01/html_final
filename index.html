O objetivo principal deste site é ensaiar rotinas de desenvolvimento web, utilizando o HTML, juntamente com CSS e JS. O conteudo principal aborda uma alimentacao saudavel para praticantes de atividades fisicas.

Treinar e fazer dieta parece fácil, mas fazer ambos corretamente e obter resultados é algo que vai bem mais além.

Aqui iremos ajudar voce a alcançar e superar resultados!

Os conteudos serão dispostos em tópicos, formando um índice agrupados com links clicaveis, para que em caso de dúvidas, possa se obter um detalhamento maior no asssunto de interesse.
